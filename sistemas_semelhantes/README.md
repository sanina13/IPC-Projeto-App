# Sistemas Semelhantes

Nesta secção são analisadas aplicações semelhantes à proposta desenvolvida no projeto.

O objetivo desta análise é identificar pontos positivos e negativos ao nível da interface e da usabilidade, permitindo retirar ideias e boas práticas para o desenvolvimento da aplicação.

Cada elemento do grupo é responsável pela análise de um sistema semelhante, garantindo assim a distribuição equilibrada do trabalho e a diversidade de perspetivas na avaliação das aplicações.
