# 💊 EasyMed — Gestão de Medicação Diária

Projeto desenvolvido no âmbito da unidade curricular de **Interação Pessoa-Computador**,
Licenciatura em Engenharia Informática — Escola Superior de Tecnologia e Gestão de Beja (IPBeja).

---

## 📋 Descrição

A **EasyMed** é uma aplicação móvel Android destinada à gestão da medicação diária.
O objetivo é ajudar utilizadores com diferentes perfis e necessidades de saúde a tomar
a medicação corretamente, no horário certo, evitando esquecimentos e doses duplicadas.
