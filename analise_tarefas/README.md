# Análise de Tarefas

Nesta secção é apresentada a análise hierárquica de tarefas (HTA) com base nos cenários de utilização definidos anteriormente.

O objetivo desta análise é decompor as tarefas realizadas pelos utilizadores em subtarefas, permitindo compreender de forma estruturada as interações necessárias para atingir os seus objetivos.

Cada elemento do grupo é responsável por desenvolver a análise de tarefas correspondente ao cenário de utilização que criou, garantindo consistência e ligação entre as diferentes fases do projeto.

Esta análise serve de base para o desenvolvimento do protótipo da aplicação, ajudando a definir a organização dos ecrãs e o fluxo de navegação.
