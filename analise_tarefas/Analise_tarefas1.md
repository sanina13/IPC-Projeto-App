# Análise de Tarefas – Cenário 1

## Gerir a toma de medicação

### 1. Receber notificação
1.1 O utilizador recebe alerta no telemóvel  
1.2 O utilizador verifica a notificação  

### 2. Aceder à aplicação
2.1 O utilizador desbloqueia o telemóvel  
2.2 O utilizador abre a aplicação EasyMed  

### 3. Consultar medicação
3.1 Visualizar o medicamento a tomar  
3.2 Verificar o horário  
3.3 Confirmar instruções (se necessário)  

### 4. Tomar medicação
4.1 O utilizador toma o medicamento  

### 5. Registar toma
5.1 Selecionar opção de confirmação  
5.2 Registar como “tomado”  

### 6. Verificar estado
6.1 Confirmar que a medicação foi registada  
6.2 Consultar histórico (opcional)
